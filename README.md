## Overview
This is a chrome tab extension to help you to be more productive. It was made as a part of Chingu voyage 11 

DOWNLOAD LINK: [CLICK HERE](https://chrome.google.com/webstore/detail/whatsnext/ameikpofcbfnfkmaeoficikekpambipg)

## Features
 - ### Search bar
      <img width="607" alt="search" src="https://user-images.githubusercontent.com/45256987/67166433-aaa00f00-f38f-11e9-8374-a36f0402499d.PNG">
      
      - Just basic search bar :grin:
      
 - ### Weather 

      <img width="250" alt="weather" src="https://user-images.githubusercontent.com/45256987/67166459-e76c0600-f38f-11e9-91d9-c3524c27dccc.PNG">

    - Weather feature with three hours intervals for 48 hours. Showing the icon of the weather, Temperature and wind speed. With ability        to choose to show temperature in celsius or in fahrenheit and ability to choose to show wind speed in km/h or m/s. It was made using [Dark Sky api](https://darksky.net/dev)


 - ### Google Calender
      <img width="292" alt="Calender" src="https://user-images.githubusercontent.com/45256987/67166604-44b48700-f391-11e9-9bf6-1ffb2470cbe4.PNG">
      
      - Calender to show your events through the day. It was made using [Google Calender api](https://developers.google.com/calendar)
 
 - ### Currency Converter
      <img width="306" alt="Currency" src="https://user-images.githubusercontent.com/45256987/67166797-1172f780-f393-11e9-9958-a17522f0410c.PNG">
      
      - Currency converter to exchange between to currencies and with ability to swap the result. It was made using [Free Currency Converter API](https://free.currencyconverterapi.com)
 
 - ### Most visited sites
 
     <img width="307" alt="sites" src="https://user-images.githubusercontent.com/45256987/67166861-c2799200-f393-11e9-9384-b8ca88bf4661.PNG">


      - Display the most visited sites by the user so user can access them faster
 - ### Countdown Timer
    <img width="291" alt="count" src="https://user-images.githubusercontent.com/45256987/67166914-4af83280-f394-11e9-942f-fe6259dcd9a8.PNG">
    
    - Countdown Timer using user input with ability to start counter ro reset it. It was made using [react-countdown-now](https://www.npmjs.com/package/react-countdown-now)
 
## Running the project
To test in development, run  
`npm run build`  
[load](https://support.google.com/chrome/a/answer/2714278?hl=en) the `build` folder as an extension.
